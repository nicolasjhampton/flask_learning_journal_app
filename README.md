# flask_learning_journal_app

To install the dependencies:

```bash
pipenv install
```

To activate the virtualenv:

```bash
pipenv shell
```

To run the program:

```bash
chmod +x run.py
./run.py
```

To run the tests:

```bash
coverage run -m unittest discover
```

To deactivate the virtualenv:

```bash
exit
```

Developed for MacOS
